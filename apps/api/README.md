# API Service

This service is the backend foundation for the Subscription Admin SaaS product.

## Responsibilities

- expose internal admin APIs
- enforce business rules
- model customer, plan, subscription, invoice, and audit workflows
- provide dashboard aggregates
- integrate with PostgreSQL and Stripe

## Planned Stack

- Node.js
- TypeScript
- Fastify
- Zod
- PostgreSQL
- Drizzle or Prisma

## Status

The repository currently contains the backend scaffold and domain contracts. Package installation and runtime setup will happen after the local Node.js toolchain is upgraded.
