# Product Overview

## Summary

Subscription Admin SaaS is an internal operations platform for businesses that need visibility and control over customers, subscriptions, invoices, and account activity.

## Target Users

- Admin
- Operations Manager
- Finance Analyst
- Support Agent

## Core Jobs To Be Done

- review business health from a dashboard
- create and manage customer accounts
- manage plans and subscription lifecycle
- follow invoice and payment status
- assign account owners
- track changes and operational history

## Differentiators For The Portfolio

- realistic B2B product framing
- multi-role internal workflow
- billing-oriented data model
- auditability and operational traceability
- table-heavy UX that mirrors real admin systems
